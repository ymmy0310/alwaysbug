# 智能小车导航系统

基于 Pygame 的路径规划与学习演示系统。

## 功能特性

- 🎨 交互式地图编辑（画笔/填充）
- 🚗 A* 路径规划
- 🧠 离线训练 + 关键点学习
- 📁 地图存档与加载

## 安装与运行

```bash
pip install pygame numpy
python car.py

应该做个bat启动的

## 操作说明

输入地图尺寸（100-300）和名称，如果输入已保存的地图名称会直接显示已绘制的地图

用画笔/填充绘制障碍物

点击「绘制完成」→ 进入移动模式 → 单击白色区域部署小车

点击「离线训练」（可选）

点击目标点，小车自动导航
